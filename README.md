# E&F-
一款英语学习的微信小程序
提供在线翻译，经验分享，话题讨论，
根据每日浏览，点赞，评论获得的积分给予当前段位
并进行积分排行
直播间功能由于时间问题目前只是静态页面 还没有全部实现

该小程序获得东北赛区一等奖
